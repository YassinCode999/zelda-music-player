
A cute Zelda-inspired music player built with React featuring a circular progress bar, spinning disk animation, and a soft aesthetic UI inspired by Hyrule 🌿✨

✨ Features
🎧 Play / Pause music control
🔵 Circular progress bar (song progress)
💿 Spinning CD-style image when playing
🎨 Zelda-inspired cute UI theme
🌿 Smooth animations
📱 Responsive design
